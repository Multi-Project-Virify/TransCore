# TransCore
TransCore — открытый локальный переводчик текста. Работает офлайн, без Google/DeepL API и без отправки текста на сервера. Определяет язык, разбивает текст и подбирает перевод по контексту. Apache 2.0, открыт для контрибьюторов.                 Если проект полезен — оставьте звезду ⭐ на GitHub, это помогает другим его найти.
