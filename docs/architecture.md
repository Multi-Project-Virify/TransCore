# Архитектура TransCore

## Принцип
Ядро (`core/`) ничего не знает о языках и переводе — оно только
прогоняет `TranslationContext` через зарегистрированные модули.

Text → Tokenizer → LanguageDetector → Dictionary → Ranking → Result

## Компоненты

- **TranslationContext** — единый объект с данными пайплайна
- **TranslationEngine** — запускает модули по очереди
- **Module** — общий интерфейс, который реализует каждый модуль

## Модули (modules/)

- `tokenizer.py` — разбивка текста на токены
- `detector.py` — определение языка (по алфавиту, локально)
- `dictionary.py` — поиск кандидатов перевода из `data/dict_en_ru.json`
- `ranking.py` — выбор лучшего перевода по контексту соседних слов

## Принцип приватности
Всё работает офлайн, локально, без внешних API и без передачи
текста пользователя куда-либо.
